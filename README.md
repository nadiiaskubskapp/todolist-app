## Simple Todo List App Docs

[![CI/CD Pipeline](https://github.com/nadiiaskubskapp/todolist-app/actions/workflows/main.yml/badge.svg)](https://github.com/nadiiaskubskapp/todolist-app/actions/workflows/main.yml)

## 1. Project Overview

TO-DOIT — це вебдодаток, розроблений для допомоги користувачам у керуванні завданнями. Він надає зручний інтерфейс для додавання, редагування та видалення завдань. Додаток також підтримує фільтрацію завдань за статусом (Усі, Очікують, Завершені) та містить перемикач тем для персоналізації користувацького досвіду.

## 2. Getting Started

Щоб почати роботу з додатком TO-DOIT, виконайте наступні кроки:

### Prerequisites

Переконайтеся, що у вас встановлено:

- Сучасний веббраузер (наприклад, Chrome, Firefox, Safari)
- Підключення до інтернету (для завантаження зовнішніх залежностей)

### Installation

1. **Клонуйте репозиторій:**

   ```bash
   git clone git@github.com:abdellatif-laghjaj/todo-list.git
   ```

2. **Перейдіть до директорії проєкту:**

   ```bash
   cd todo-list
   ```

3. **Відкрийте index.html у браузері:**

   Відкрийте файл index.html у вашому улюбленому веббраузері.

## 3. Usage

1. **Додавання завдання:**
   - Введіть завдання в поле введення.
   - За бажанням встановіть дату виконання за допомогою календаря.
   - Натисніть Enter або кнопку «+», щоб додати завдання.

2. **Редагування завдання:**
   - Натисніть кнопку «Редагувати» (Edit) на завданні.
   - Змініть деталі завдання.
   - Натисніть кнопку з галочкою (Check), щоб зберегти зміни.

3. **Завершення завдання:**
   - Натисніть кнопку з галочкою на завданні, щоб змінити його статус завершення.

4. **Видалення завдання:**
   - Натисніть кнопку з кошиком (Trash) на завданні, щоб видалити його.

5. **Фільтрація завдань:**
   - Використовуйте випадний список «Фільтр» (Filter), щоб відфільтрувати завдання за статусом (Усі, Очікують, Завершені).

6. **Очищення всіх завдань:**
   - Натисніть кнопку «Видалити все» (Delete All), щоб видалити всі завдання.

7. **Перемикання тем:**
   - Використовуйте іконку палітри у верхньому правому куті, щоб відкрити перемикач тем.
   - Виберіть тему із запропонованих варіантів.
